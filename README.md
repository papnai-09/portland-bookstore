# Portland Bookstore Author Reading

A professional and modern website for Portland Bookstore's Author Reading Night event.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, professional interface with smooth animations
- **Author Showcase** - Beautiful card-based layout for featured authors
- **Event Registration** - Call-to-action button for event registration
- **Accessible** - Built with semantic HTML and ARIA attributes

## File Structure

```
/
├── index.html          # Main HTML page
├── css/
│   └── style.css       # Styling and responsive design
└── README.md          # This file
```

## Technologies Used

- HTML5
- CSS3 (with CSS Variables, Flexbox, Grid)
- Modern Web Standards

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize author information and styling as needed

## Customization

### Update Author Information

Edit the author cards in `index.html` to include your own author details.

### Modify Colors

Update the CSS variables in `css/style.css`:

```css
:root {
    --brand-primary: #8B0000; /* Deep Red */
    --brand-secondary: #FFF5E1; /* Cream */
    --text-dark: #1a1a1a;
    --white: #ffffff;
}
```
